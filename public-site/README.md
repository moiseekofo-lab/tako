# Mini site public TaKo

Ce dossier contient le mini site public de liens TaKo.

## Déploiement Render

- Type : Static Site
- Root Directory : `public-site`
- Build Command : laisser vide
- Publish Directory : `.`

Le domaine public peut ensuite pointer vers ce service Render.

## Liens à personnaliser

Dans `index.html`, remplacez les liens WhatsApp `https://wa.me/243000000000` par le vrai numéro TaKo.
